8. 一维数组的内存解析
8.1 Java中的内存结构是如何划分的？（主要关心JVM的运行时内存环境）
应该叫做runtime data area
共享的： 堆，方法区
线程安全：程序计数器（Program counter register)/ native method stack / java virtual stack


8.2 举例：具体一维数组的代码的内存解析

int[] arr = new int[]{1,2,3};
>虚拟机栈：存放方法中声明的变量，比如arr
>堆：存放数据的实体，比如1,2,3
